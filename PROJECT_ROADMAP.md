### 🔵 ФАЗА 1 — Полировка прототипа
*Цель: сделать игру приятной и законченной как веб-приложение*

- [ ] Добавить щит при старте уровня (2 сек неуязвимости)
- [ ] Настроить баланс уровней (скорость, количество еды, препятствия)
- [ ] Добавить анимацию «Level Up!» при переходе уровня

#### 1.2 Звук и музыка
- [ ] Заменить внешний MP3 файл на встроенную процедурную музыку (Web Audio)
- [ ] Добавить звук удара о стену / препятствие

#### 1.3 Визуал и UI

### 🟡 ФАЗА 2 — Контент и глубина
*Цель: игрок хочет возвращаться снова и снова*

#### 2.1 Режимы игры
- [ ] **Классика** — бесконечная игра, растущая скорость (уже почти есть)
- [ ] **Время** — успей набрать максимум за 60 секунд
- [ ] **Выживание** — максимум препятствий, скорость не растёт
- [ ] **Зеркало** — управление инвертировано (влево = вправо)

#### 2.2 Бонусные предметы
- [ ] ⭐ Золотая еда — появляется на 5 сек, даёт x3 очков
- [ ] 🛡️ Щит — следующее столкновение не убивает
- [ ] ⚡ Ускорение — временный буст скорости
- [ ] ❄️ Заморозка — замедляет змейку на 3 сек
- [ ] 💣 Бомба — уничтожает все препятствия на поле

#### 2.3 Достижения
- [ ] «Первая еда», «10 в ряд», «Уровень 10», «Без урона», «Спидран» и др.
- [ ] Экран достижений с прогрессом

#### 2.4 Магазин скинов (без реальных денег пока)
- [ ] Покупка скинов за игровые монеты
- [ ] Монеты начисляются за игру (1 монета = 1 очко)

## 🛠️ ТЕХНОЛОГИЧЕСКИЙ СТЕК

| Что | Инструмент | Зачем |
|-----|-----------|-------|
| Игровой движок | HTML5 Canvas + JavaScript | Работает везде, не нужны сторонние библиотеки |
| Звук | Web Audio API | Встроен в браузер, не нужны файлы |
| Хранение данных | localStorage → Firebase | Локально сейчас, онлайн потом |
| Публикация мобайл | Capacitor.js | Превращает HTML в iOS/Android приложение |
| Монетизация iOS | Apple StoreKit | Встроенные покупки для App Store |
| Монетизация Android | Google Billing | Встроенные покупки для Play Store |
| Реклама | Google AdMob | Самая популярная рекламная сеть для мобильных |
| Хостинг (веб) | GitHub Pages | Бесплатно, уже используется |
| Лидерборд | Firebase Firestore | Бесплатный онлайн-рекорд для начала |
