# 🐍 Snake

A modern Snake game built with pure HTML5 Canvas and Web Audio API.

**[▶ Play Now](https://youngsanifone.github.io/snake/)**

---

## Features

- Smooth canvas-based movement with delta-time physics
- 5 unlockable skins — new ones appear as you level up
- Procedural music + sound effects (Web Audio API)
- Combo system — eat food in a row for bonus points
- Gold food ⭐, bonus items 🛡⚡❄💣
- Responsive design — works on mobile, tablet, and desktop
- Touch controls (swipe or D-pad) + keyboard support

## Controls

| Input | Action |
|-------|--------|
| ↑ ↓ ← → / WASD | Move |
| Space / D-pad center | Pause |
| Swipe | Move (mobile) |

## Tech

Built with vanilla JavaScript — no frameworks, no dependencies.

- HTML5 Canvas 2D
- Web Audio API
- localStorage
